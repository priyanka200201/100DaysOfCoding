QUESTION:  Majority Element II
//https://leetcode.com/problems/majority-element-ii/description/

Given an integer array of size n, find all elements that appear more than  n/3 times.

Test Case 1:

Input: nums = [3,2,3]
Output: [3]

Test Case 2:

Input: nums = [1]
Output: [1]

Test Case 3:

Input: nums = [1,2]
Output: [1,2]
 

Constraints:

1 <= nums.length <= 5 * 104
-109 <= nums[i] <= 109
